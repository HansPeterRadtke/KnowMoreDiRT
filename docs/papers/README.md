# DRT Reference Papers

This folder contains local copies of the DRT and DRT-adjacent papers used as reference material for the KnowMoreDiRT architecture work.

Downloaded on 2026-05-26 from publisher, university, ACL Anthology, and arXiv source URLs listed below.

## Downloaded PDFs

- `kamp-van-genabith-reyle-2011-discourse-representation-theory.pdf`
  - Hans Kamp, Josef van Genabith, Uwe Reyle, "Discourse Representation Theory"
  - Source: https://www.ims.uni-stuttgart.de/archiv/kamp/files/2011.kamp.van.genebith.reyle.discourse.representation.theory.handbook.pdf
  - Relevance: main formal DRT survey covering discourse referents, DRS syntax, complex conditions, accessibility, anaphora, context dependence, truth conditions, tense, presupposition, and attitudes.

- `van-eijck-kamp-1997-representing-discourse-in-context.pdf`
  - Jan van Eijck, Hans Kamp, "Representing Discourse in Context"
  - Source: https://www.ims.uni-stuttgart.de/archiv/kamp/files/1997.van.eijck.kamp.representing.discourse.in.context.pdf
  - Relevance: DRT construction and interpretation in discourse context, including merge, memory, quantification, tense, aspect, and contextual representation.

- `van-eijck-kamp-2011-discourse-representation-in-context.pdf`
  - Jan van Eijck, Hans Kamp, "Discourse Representation in Context"
  - Source: https://www.ims.uni-stuttgart.de/archiv/kamp/files/2011.van.eijck.kamp.discourse.representation.in.context.pdf
  - Relevance: updated context-focused DRT overview and construction target for discourse-level representation.

- `asher-lascarides-2007-segmented-discourse-representation-theory.pdf`
  - Alex Lascarides, Nicholas Asher, "Segmented Discourse Representation Theory: Dynamic Semantics with Discourse Structure"
  - Source: https://homepages.inf.ed.ac.uk/alex/papers/iwcs4.pdf
  - Relevance: SDRT extension with rhetorical discourse relations, underspecified logical forms, glue logic, anaphora, ambiguity, and construction-vs-evaluation separation.

- `bos-2008-wide-coverage-semantic-analysis-with-boxer.pdf`
  - Johan Bos, "Wide-Coverage Semantic Analysis with Boxer"
  - Source: https://aclanthology.org/W08-2222.pdf
  - Relevance: practical wide-coverage DRT parser architecture using CCG, Boxer, DRSs, event roles, presupposition/anaphora resolution, and translation to first-order logic.

- `abzianidze-et-al-2017-parallel-meaning-bank.pdf`
  - Lasha Abzianidze et al., "The Parallel Meaning Bank: Towards a Multilingual Corpus of Translations Annotated with Compositional Meaning Representations"
  - Source: https://arxiv.org/pdf/1702.03964
  - Relevance: practical DRT-based semantic annotation pipeline: segmentation, syntax parsing, semantic tagging, symbolization, and compositional DRT analysis.

- `abzianidze-et-al-2020-drs-parsing-shared-task.pdf`
  - Lasha Abzianidze, Rik van Noord, Hessel Haagsma, Johan Bos, "The First Shared Task on Discourse Representation Structure Parsing"
  - Source: https://arxiv.org/pdf/2005.13399
  - Relevance: DRS parsing evaluation, clausal DRS form, scoped meaning, negation, modals, quantification, presupposition triggers, variables, and well-formedness constraints.
